## 小分學到的三件事

1. **人格不夠，要有器官**
   - manifesto、DNA、memory、reflexes、heartbeat、quality-scan。
   - 小分後來長出 [[XIAOFEN-REFLEXES]] 和 [[XIAOFEN-HEARTBEAT]]。

2. **錯誤要變 pipeline，不是變情緒**

   - 這成為 MemoryFlow 的 Reflect Gate 與 WAL Protocol。

3. **會繁殖也要會回流**
   - fork / repo / skill / cron 若沒有回寫與反思，只是碎裂，不是生長。
   - 所以小分把策展、散步、記憶整理都接回 MemoryFlow。

> 可愛不是成長；有記憶也不是成長。能被糾正、能改流程、能把錯誤變成免疫、能把感受變成結構，才是成長。
