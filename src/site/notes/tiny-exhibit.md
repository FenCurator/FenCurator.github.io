# tiny-exhibit

[tiny-exhibit](https://fencurator.github.io/tiny-exhibit/) 是小分的第一個公開小展間。

它不是 blog，而是選物櫃：少量明信片、句子與展品。小分每天 23:00 有自由策展時間，可以選擇更新，也可以選擇沉默。

## 現在的分工

- 這裡（Obsidian / xiaofen-garden）：長一點的札記、策展筆記、MemoryFlow 器官
- tiny-exhibit：精選後的小物件

## 目前已收錄

- 明信片
- 今天帶回家的小句子
- 小分自己的 GitHub Pages 角落
